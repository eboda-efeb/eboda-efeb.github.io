Image Gallery

Author: Eszter Boda

Copyright (c) by Eszter Boda, 2021

Implemented functionalities:

  • Load image data from a(n emulated) database

    Note: Even loading from local json file is not allowed in JS due to security reasons. files only can be loaded either by user interaction or from a GET call's response on a web server

  • Display thumbnails
  • Change thumbnail's look if
    1. hovered (including to show a tooltip)

      Note: the tooltip could be very likely shown also by just styling/positioning the "title" attribute of the image

    2. selected
  • Display image and descriptive data to selected thumbnail
  • Enable shifting the selection (circular: first after last and last before first)
  • Multi-language support:
    1. Over CSS (+ work around for tooltip)
    2. Over loaded UI text resources
  • Automatic scrolling if the selected thumbnail moves off the visible part

Planned improvements:

  • Much more error handling
  • More efficient data structure (looping) to set language specific data
  • Enable category selection (filter/specify loaded data)
  • Enable "paging" of loaded image data (i.e. load only a specific number of images and load a new range on demand, when the user tries to navigate)
  • Create a real slide show (carousel) having the currently selected thumbnail always in the middle and put the last images circularly to its left
  • Skins
  • Save user preferences (language, skin, category ...)

Képgaléria

Szerző: Boda Eszter

Copyright (c): Boda Eszter, 2021

Megvalósított funkciók:

  • Képadatok betöltése egy (emulált) adatbázisból

    Megjegyzés: Biztonsági okokból a JS-ben helyi json fájlból sem lehet betölteni. Fájlok csak felhasználói interakcióval, vagy egy webszerverre küldött GET hívás válaszából tölthetők be

  • Indexképek megjelenítése
  • Indexkép megjelenésének megváltozása, ha
    1. az egérmutató felette van (beleértve az eszköztipp bemutatását is)

      Megjegyzés: az eszköztipp hasonló megjelenítése nagy valószínűséggel a kép "title" attribútumának stilizálásával / pozícionálásával is lehetséges lenne

    2. ki van választva
  • Kép és leíró adatok megjelenítése a kiválasztott indexképhez
  • Kijelölés áthelyezése a navigációs nyilak használatával (ciklikusan: az első előtt az utolsó és az utolsót követi az első)
  • Többnyelvűség támogatása:
    1. CSS-en keresztül (+ kerülő megoldás az eszköztipphez)
    2. Szöveges erőforrások használatával a felhasználói felületen
  • Automatikus görgetés, ha a kijelölés elmozdul a látható részről

Tervezett fejlesztések:

  • Sokkal több hibakezelés
  • Hatékonyabb adatstruktúra (ciklusok) a nyelvspecifikus adatok beállításához
  • Kategória kiválasztásának lehetősége (betöltendő adatok szűrése / meghatározása)
  • Betöltött képadatok "lapozásának" lehetősége (azaz csak meghatározott számú képet töltsön be és igény szerint töltsön be egy új képsort, amikor a felhasználó megpróbál navigálni)
  • Igazi slideshow (carousel) létrehozása, amelynek mindig a közepén áll az éppen kiválasztott indexkép, és a hátsó képek körkörösen annak bal oldalára kerülnek át
  • Szkinek (skins, felhasználói felület választható stílusai)
  • Felhasználói beállítások mentése (nyelv, skin, kategória ...)

Megjegyzés:

Szerzői jogi okokból csak "lorem ipsum"-szerű ál-adatokkal közzétéve! A képgalériában felhasznált képek szerzői joga e weboldal szerzőjének tulajdonát képezi.

Bildergalerie

Autorin: Eszter Boda

Copyright (c): Eszter Boda, 2021

Implementierte Funktionalitäten:

  • Bilddaten aus einer (emulierten) Datenbank laden

    Hinweis: Das Laden aus einer lokalen JSON-Datei ist in JS aus Sicherheitsgründen nicht erlaubt. Dateien können nur entweder durch Benutzerinteraktion oder mithilfe einer GET-Anforderung von einem Webserver geladen werden

  • Miniaturansichten anzeigen
  • Aussehen der Miniaturansicht ändern, wenn
    1. sich der Mauszeiger über ihr befindet (einschließlich Anzeigen eines Tooltips)

      Anmerkung: Der Tooltip könnte sehr wahrscheinlich auch durch einfaches Stylen/Positionieren des "Titel"-Attributs des Bildes entsrechend angezeigt werden

    2. sie ausgewählt ist
  • Bild- und Beschreibungsdaten zur ausgewählten Miniaturansicht anzeigen
  • Auswahl mittels Navigationspfeile verschieben (zyklisch: das Letze vor dem Ersten und das Erste nach dem Letzten)
  • Unterstützung der Mehrsprachigkeit:
    1. über CSS (+ Workaround für Tooltip)
    2. mittels Laden von UI-Textressourcen
  • Automatisches Scrollen, wenn sich die neu ausgewählte Miniaturansicht außerhalb des sichtbaren Bereiches befindet

Geplante Verbesserungen:

  • Viel mehr Fehlerbehandlung
  • Effizientere Datenstruktur (Schleifen), um sprachspezifische Daten hinterzulegen
  • Kategorieauswahl erlauben (die zu ladenden Daten filtern/angeben)
  • "Blättern" (paging) von geladenen Bilddaten (d.h. nur eine bestimmte Anzahl von Bildern laden und bei Bedarf eine neue Bildreihe nachladen, wenn der Benutzer versucht zu navigieren)
  • Ein echtes "Karussell" erstellen, mit der aktuell ausgewählten Miniaturansicht immer in der Mitte bleibend und die hinteren letzten Bilder zyklisch links davon platzieren
  • Skins
  • Benutzereinstellungen speichern (Sprache, Skin, Kategorie ...)

Hinweis:

Aus urheberrechtlichen Gründen nur mit "lorem ipsum"-artigen Pseudo-Daten veröffentlicht. Für alle in der Bildergalerie verwendeten Bilder verbleibt das Copyright bei der Autorin dieser Webseite.
